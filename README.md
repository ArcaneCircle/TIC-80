# TIC-80 for Webxdc

A [Webxdc](https://webxdc.org) port of the [TIC-80 TINY COMPUTER](https://tic80.com),
a free and open source fantasy computer for making, playing and sharing tiny games.

With this port you can share the TIC-80 computer inside chats
and import/export files via add/get commands.
